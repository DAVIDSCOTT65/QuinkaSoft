L'application QStock est monoposte cad, fonctionne sur un seul PC

Vous devez creer un dossier dans le disque C et le renomé "QSTOCK DATABASE"
 N'effacer jamais ce dossier sinon l'application ne fonctionnera pas.

Veuilez garder hebdomaderairement en ligne sur google drive le fichier .mdf contenus dans ce dossier.
